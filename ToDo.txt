User->RestAPI->Load_balancer->5*Service->Service_db.
4. Configure application servers using Docker according to the previous
schema.
5. On RestAPI server create `routes.php` file to map specified endpoints to
configured servers.
6. Create template php files on corresponding microservices.
7. Create databases according to the schema.
8. Fill the databases with sample data.
9. Program CRUD operations on sample data, display results in the browser.
